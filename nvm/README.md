View NVM Version:
```
nvm -v
```

Install Latest Node.js:
```
nvm install latest
```

Install Node.js by version:
```
nvm install vX.Y.Z
```

Make a version default:
```
nvm alias default vX.Y.Z
```

Use a node version:
```
nvm use vX.Y.Z
```
